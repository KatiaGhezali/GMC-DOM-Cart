# GMC-DOM-Cart

A simple shopping cartr application built with **HTML**, **CSS**, and **JavaScript DOM manipulation**.

## Features

- Increase item quantity
- Decrease item quantity
- Remove items from the cart
- Like/Unlike items
- Automatically update the total price

## Technologies

- HTML5
- CSS3
- JavaScript (DOM Events)
